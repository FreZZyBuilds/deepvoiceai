## 🚀 Quick Start

```bash
cd src
npm start          # Run React frontend (localhost:3000)
```

Backend:
```bash
cd src
pip install -r requirements.txt
python app.py      # Run Flask API (localhost:5000)
```

Tests:
```bash
pytest testsprite_tests/ -v
```
